Minha Biblioteca: Mil vidas em apenas uma!

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma coleção de livros mais vendidos. O objetivo é criar um ambiente virtual onde seja possível vender, liveos de aurores conhecidos e proporcionar uma leitura digital, com parceria da @Amazon além de fornecer uma opção de compra.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de livros em uma seção flexível (.livros). Cada livro é destacado em seu próprio contêiner (.livro), mostrando uma imagem da capa, detalhes do livro e um link de compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração da coleção de livros.

Meus amigos

O projeto consiste em uma web relacionada aos meus amigos, desenvolvida com HTML, CSS, e JavaScript, que proporcionar uma experiência de como conheci o meus amigos. O objetivo é proporcionar para os visitantes do site um lugar aconchegante com histórias de como eu conheci eles e personagens de acordo com a personalidade de cada um!

O HTML estrutura o conteúdo da página, dividindo-o em seções como a tela inicial, divisão de acordo com cada personagem. O CSS é utilizado para estilizar a página, empregando um esquema de cores futurista e layout responsivo para melhorar a experiência do usuário. JavaScript é usado para adicionarinteratividade com os personagens, manipulando o DOM para atualizar o conteúdo da página.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a apresentação dos meus friends em uma seção flexível. Cada livro é destacado em seu próprio contêiner (.friends), mostrando uma imagem da capa e personagens. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

Os scripts JavaScript incluem módulos para gerar nomes, definir os personagens e as apresentações. Este design modular facilita a manutenção e a organização do site. A importância das ferramentas utilizadas (HTML, CSS, JavaScript) reside na criação de uma experiência interativa.

Culinária

O projeto é uma aplicação educativa interativa focada em ensinar conceitos básicos de gastronomia. Utiliza uma série de scripts associados a diferentes tópicos(sprites) e cenários para criar uma experiência de aprendizado envolvente. O objetivo é familiarizar os usuários com tópicos sobre receitas, chefs, indicações de diferentes pratos, e a diversidade de culturas, através de interações e visualizações dinâmicas.

Cada tópico no projeto possui scripts que definem seu comportamento em resposta a perguntas específicas, como o início do programa (quando o card é tocado), cliques do usuário. Estes eventos desencadeiam ações como mudanças de cenário,exibição ou ocultação das cartas, e a apresentação de diálogos informativos. Por exemplo, ao clicar em determinados tópicos, os usuários podem "receber" uma prova, o que leva a uma mudança de cenário e a exibição de informações relevantes ou a visualização de alguma receita.

A importância das ferramentas utilizadas, como html e style, reside na sua acessibilidade e na capacidade de engajar aprendizes de diferentes idades no pensamento computacional e na exploração de relacionados a gastronomia de forma interativa. O uso de cenários, movimentos, e diálogos dinâmicos ajuda a criar uma experiência de aprendizado mais rica e memorável, resultando em algo atrativo para os usuários e melhor acessibilidade.
